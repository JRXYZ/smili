smili.geomodel.geomodel.Gaussian(x0=0, y0=0, totalflux=1, majsize=1, minsize=None, pa=0, angunit=u'mas')¶

Create geomodel.geomodel.GeoModel Object for the specified Gaussian :param x0=0.0, y0=0.0: position of the centorid :param totalflux=1.0: total flux of the Gaussian :param majsize, minsize: Major/Minor-axis FWHM size of the Gaussian :param pa: Position Angle of the Gaussian in degree :param angunit=”mas”: angular unit of x0, y0, majsize, minsize (uas, mas, asec, amin, deg, rad)

Returns: :returns: geomodel.geomodel.GeoModel Object for the specified Gaussian
